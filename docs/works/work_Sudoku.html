Design

WFC behaviour

This Sudoku works with a WFC (wave function collapse) algorithm.
The definition of WFC is: "An algorithm inspired by the ideas of quantum mechanics. According to one of such ideas, there's a huge set of possible states for each object. Initially, an object has some generalized state represented as a linear combination of all the elements of this set."

Sudoku?

This classic game is a very good example of WFC behaviour. When you choose a number in one cell, automatically you are elminating possibilities from another cells.
With this prototype, I managed to learn how WFC works and to implement it in a practical way in Unity. I am sure that this procedural generation will be useful in my future projects.
